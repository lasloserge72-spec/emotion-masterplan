# Rough-In Electrical Master Plan

Date: 05.05.2026  
Object: two-storey villa with accessible roof garden  
Location: Batumi, Georgia  
Building area: 303.42 m²  
Roof Garden: 108.50 m²  
Stage: rough electrical works, cable routes, boards, sleeves, low-voltage lines

## System Scope

| Section | Scope |
|---|---|
| Power entry | incoming cable, main breaker, meter, entry board |
| Distribution | main board, floor groups, breakers, RCD/differential breakers |
| Lighting | cable lines, switches, two-way switching, outdoor lighting |
| Socket groups | general sockets, kitchen circuits, bathrooms, bedrooms, living/dining |
| Power circuits | kitchen, water heater/technical unit, AC units, pumps, outdoor units |
| Low voltage | internet, Wi-Fi points, TV, cameras, intercom, sensors |
| Earthing | grounding loop, PE busbar, equipotential bonding |
| Lightning protection | down conductors, grounding, equipment protection |
| Outdoor electrical | IP65 lines, waterproof boxes, separate protected groups |
| Reserve lines | sleeves and spare lines for smart home, security, landscape |

## Electrical Boards

| Board | Location | Purpose |
|---|---|---|
| Entry board | technical / entrance area | incoming power, metering, main breaker, surge protection |
| Main distribution board | first floor | main house distribution |
| Floor board | second floor | lighting, sockets, bathrooms, bedrooms |
| Outdoor board | roof garden / technical zone | outdoor units, lighting, sockets, bar, pumps |
| Low-voltage cabinet | first floor / technical zone | router, switch, NVR, patch panel, low-voltage power |
