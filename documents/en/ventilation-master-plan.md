# Ventilation Master Plan

Date: 05.05.2026  
Object: two-storey villa with accessible roof garden  
Location: Batumi, Georgia  
Building area: 303.42 m²  
Roof Garden: 108.50 m²  
Stage: rough ventilation routes, exhaust ducts, supply lines, sleeves

## System Scope

| Section | Scope |
|---|---|
| Exhaust ventilation | bathrooms, washrooms, kitchens, walk-in closets |
| Supply ventilation | living/dining, bedrooms, TV room, hall |
| Kitchen extraction | separate ducts from kitchen and second kitchen |
| Dehumidification | wet zones, coastal climate, humidity control mode |
| Air conditioning | split units, refrigerant lines, drainage |
| Roof Garden | bar-zone ventilation and service sleeves |
| Vent stacks | sewer vent stacks above roof level |
| Automation | humidity sensors, timers, fan control |

## Ducts and Air Routes

| Section | Execution |
|---|---|
| First floor bathrooms | exhaust ducts to vertical riser |
| Second floor bathrooms | exhaust ducts to roof outlet |
| Washrooms | separate exhaust ducts with humidity sensors |
| Kitchen | separate grease-resistant hood duct |
| Second kitchen | separate exhaust duct |
| Bedrooms | supply lines or wall/ducted supply devices |
| Living/dining | supply lines, transfer to exhaust zones |
| Roof Garden | outlets above occupied zones |

| Washroom exhaust | DN125-DN160 |

## Air Conditioning

|---|---|
