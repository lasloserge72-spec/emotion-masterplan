## Structural System

| Element | Solution |
|---|---|
| Load-bearing system | cast-in-place reinforced concrete frame |
| Floor slabs | cast-in-place reinforced concrete slabs |
| Spatial stiffness | reinforced concrete walls/cores in calculated zones |
| External walls | block infill within the frame |
| Internal partitions | block/lightweight partitions |
| Roof garden | heavy accessible roof assembly |
| External stair | calculated fixings and support nodes |
| Glass railings | calculated base fixings |
| Facade substructure | calculated fixing to load-bearing base |

## Foundation

| Option | Scope | Cost per house |
|---|---|---:|
| Monolithic raft slab | blinding concrete, crushed stone base, waterproofing, 250-350 mm slab, local strengthening, drainage | 75,000-110,000 GEL |
| Pile-raft / pile-grillage | bored piles, grillage or slab on piles, reinforced waterproofing, drainage | 115,000-180,000 GEL |
| Complex slope | retaining walls, stepped preparation, drainage, geotextile, backfill | +40,000-120,000 GEL |

Base calculation for the series uses a monolithic raft slab.

## Facade System

| Zone | Execution |
|---|---|
| White surfaces | reinforced facade layer + silicone/siloxane render + hydrophobic coating |
| Dark areas | 120 x 60 cm porcelain tiles on ventilated substructure |
| Windows | thermally broken aluminum profile, warm installation, sealing tapes, flashings |
| Railings | tempered/laminated glass, A4/316 stainless steel base |
| Horizontal projections | aluminum caps with slope, drip edge, sealed joints |
| Fixings | A4/316 stainless steel or hot-dip galvanized |

## Engineering Nodes

| Node | Execution |
|---|---|
| Parapets | metal caps with slope, double drip edge, sealed joints |
| Roof garden | main drains + emergency overflows |
| AC condensate | organized discharge to stormwater/drainage |
| Window junctions | warm installation, internal/external tapes, flashings |
| Wet zones | reinforced waterproofing for bathrooms, kitchens, bar, roof garden |
| Ventilation | exhaust, supply, dehumidification mode, humidity sensors |
| Access panels | drains, manifolds, roof drains, condenser units |
| Outdoor electrical | IP65, separate breakers/RCDs, waterproof boxes |
| Roof garden finish | 20 mm porcelain pavers on adjustable pedestals |

Engineering node cost: **+90,000-120,000 GEL** per house.
