# Эмоция — мастер-план посёлка

Концепт-проект посёлка 50 домов в Хелвачаури / Шарабидзееби, Аджара, Грузия.

**Координаты:** 41,581612° N, 41,677857° E
**Площадь:** 3,98 га (39 848 м²)
**Перепад:** 104 м (205 → 309 м БС)

## Карты

- **3D-карта Mapbox** — [`MASTERPLAN_MAPBOX.html`](./MASTERPLAN_MAPBOX.html)
- **2D-карта Leaflet** — [`MASTERPLAN_INTERACTIVE.html`](./MASTERPLAN_INTERACTIVE.html)
- **Лендинг** — [`index.html`](./index.html)

## Данные

- `masterplan_4326.geojson` — сводный слой 140 объектов в WGS84
- `parcels_khelvachauri.geojson` — кадастровые границы 4 наделов

## Сводные показатели

- 50 жилых лотов × 335 м² (19 на насыпных платформах +2,5 м)
- 5 общественных платформ + велнес-павильон 200 м² + 5 купален
- 2 пруда для рыбалки членами посёлка
- 9 дорожных осей (928,5 м), 4 разворота R=9 м
- 2 КТП × 400 кВА, 34 опоры освещения LED
- Источники воды: восточный ручей + скважина 80–120 м (этап 1)
- Σ выемка / насыпь: 44 500 / 44 300 м³ (no-export)

Стадия: предпроектный концепт. Финальный проект — после топосъёмки 1:500.
