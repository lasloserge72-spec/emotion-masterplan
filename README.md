# Эмоция — мастер-план посёлка

Концепт-проект посёлка 50 домов в Хелвачаури / Шарабидзееби, Аджара, Грузия.

**Координаты:** 41,581612° N, 41,677857° E
**Площадь:** 3,98 га (39 848 м²)
**Перепад:** 104 м · 5 террас 227–280 м БС

## Версия v3 «Чистая» (2026-05-06)

- 50/50 жилых лотов, все ≥350 м² (средняя 352, мин 350, макс 360)
- Фронт ≥12 м (мин 13 м)
- Распределение по террасам: T240=6, T255=23, T270=15, T280=6
- Дороги 4 м: 981 м линий + 408 м² разворотов/карманов
- Нежилое 2 120 м²: велнес 780, ресторан 520, парковка 420, 2 ТП, 6 септиков, ТБО, сервис УК, пикник
- 4 точки доступа: AP4 (запад), AP5 (восток), AP6 (юг главный), AP_E (юго-восток)
- 3 подпорные стены: ЖБ кантилевер 189 м (h=15), габион 69+105 м (h=11)

## GeoJSON слои (EPSG:4326)

- `site_4326.geojson` — граница участка
- `engineering_4326.geojson` — 5 террас, 3 стены, 3 коридора, талвег, 4 AP
- `municipal_road_4326.geojson` — внешняя бетонная дорога
- `roads_4326.geojson` — внутренние дороги (5 улиц + ветви + развороты)
- `nonres_4326.geojson` — 14 объектов нежилого
- `lots_4326.geojson` — 50 жилых лотов
- `grid_4326.geojson` — служебная сетка 1/5/10/50 м
- `parcels_khelvachauri.geojson` — кадастровые границы 4 наделов
- `meta.json` — нулевая точка, bbox, перечень террас

Исходник в EPSG:32638 (UTM 38N), узлы целочисленной сетки 1 м.
